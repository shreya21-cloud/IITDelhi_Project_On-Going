# AI Research Paper Study Notes

## Overview

This repository contains my personal study notes and understanding of various AI research papers explored during my internship learning journey. The purpose of this repository is to document key concepts, architectures, methodologies, and insights from state-of-the-art research in Computer Vision, Vision-Language Models, Multimodal Learning, and Large Language Models.

Rather than simply collecting papers, I focused on understanding how these models work, why they were developed, their training processes, and their real-world applications.

## Papers Studied

### CLIP (Contrastive Language–Image Pretraining)

* Understanding image and text encoders
* Shared embedding space
* Contrastive learning and cosine similarity
* Zero-shot classification
* Applications in computer vision and multimodal AI

### LLaVA (Large Language and Vision Assistant)

* Vision-language instruction tuning
* Integration of CLIP with Large Language Models
* Multimodal conversation capabilities
* Training pipeline and architecture
* Visual understanding and reasoning

### Grasp-CLIP

* Language-guided robotic grasping
* CLIP-based feature fusion
* 6-DoF grasp detection
* Object selection in cluttered environments
* Robotic manipulation using multimodal understanding

### AA-CLIP

* Anomaly-aware feature learning
* Zero-shot anomaly detection
* Industrial inspection applications
* Improved representation learning for abnormal samples

 # Multimodal Product Choice Personalization

**Status:** 🚧 Ongoing

This is an ongoing AI/ML project focused on building a multimodal product choice personalization system using visual and textual information. The project aims to learn user preferences by combining image and text embeddings with historical interaction data to generate personalized product recommendations. The current work includes data preprocessing, preference dataset creation, train-test split, embedding generation, and preference scoring. Further development will focus on multimodal model integration, recommendation optimization, and performance evaluation.
